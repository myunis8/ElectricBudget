Sincronización entre dos personas

Esto guarda tus proyectos y precios en un repositorio de GitHub compartido, para que la otra persona vea lo mismo que vos. Cada uno carga acá su propia clave (token) — nunca se comparte el token, solo los datos.
1. Creá el repositorio (una sola vez, lo hace cualquiera de los dos):
Entrá a github.com → botón verde "New" → nombre por ejemplo presupuestos-datos → marcá "Private" → "Create repository".

2. Invitá a la otra persona:
Dentro del repositorio → Settings → Collaborators → "Add people" → buscála por su usuario de GitHub. Ella tiene que aceptar la invitación (le llega por mail o en github.com/notifications).

3. Cada uno genera su propio token:
github.com → foto de perfil → Settings → (al final del menú izquierdo) Developer settings → Personal access tokens → Fine-grained tokens → "Generate new token" → elegí el repositorio presupuestos-datos → en "Repository permissions" poné "Contents: Read and write" → Generate. Copiá el token (empieza con github_pat_ o ghp_) y pegalo acá arriba.

Usuario y repositorio son los mismos para los dos (el dueño del repositorio y su nombre). El token es distinto para cada persona y no hay que compartirlo.
